# Release Notes - v1.22.0

## Overview

This release focuses on enhancing the observability and detail of model configuration within the SDK examples and documentation components.

## Detailed Changes

### Improvements

- **Enhanced Model Metadata in Examples**: Updated the `newKronk` function across all major examples to provide a more comprehensive view of model configuration. New details printed include:
  - Flash Attention status
  - `nSeqMax` (Parallel Inference)
  - Total VRAM (MiB)
  - Slot Memory (MiB)
  - Model Size (MB)
  - System Prompt Cache (SPC) status
  - Incremental Message Cache (IMC) status
  - GPU Layer count (`nGPULayers`)

### Documentation & Examples

- **SDK Examples**: Updated `DocsSDKExamples.tsx` in the BUI to reflect the enhanced metadata format.
- **Refactored Example Config**: Updated `examples/chat/main.go` to use `ModelID` for model specification instead of `SourceURL`, following best practices.
