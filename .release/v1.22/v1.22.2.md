# Release Notes - 1.22.2

## Overview

This release includes various fixes and enhancements, with a focus on Gemma 4 support, IMC improvements, and VRAM calculator enhancements.

## Detailed Changes

### Features

- **Gemma 4 Support**: Added support for Gemma 4 models and addressed related fixes.
- **Model Configuration**: New functionality to analyze model configuration.
- **VRAM Calculator Enhancements**: Improved the VRAM calculator with better caching and shorthand URL support.
- **Hardware Auto-detection**: Added automatic GPU hardware detection.
- **Template Support**: Model configuration can now define templates.

### Fixes

- **IMC Improvements**: Fixed a data race in IMC and addressed other IMC-related issues.
- **API Fixes**: Fixed issues with the Responses API and tool calling.
- **Linter/Build Fixes**: Resolved linter issues and testing during release.
- **Nix Support**: Fixed dependency SHA and improved Nix support.
- **General Fixes**: Minor fixes for the goose CLI and manual improvements.

### Other Changes

- Upgraded dependencies.
- Changed default port to 11435.
- Improved BUI (Browser UI) and cache VRAM calculator model info.
- Updated workshop and examples.
