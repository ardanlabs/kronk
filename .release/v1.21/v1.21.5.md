# Release Notes - v1.21.5

**Release Date:** April 2, 2026

## Overview

Version 1.21.5 introduces automatic GPU hardware detection, changes the default server port to 11435, and improves the Nix build system with centralized version management. This release also includes dependency updates and Kilo client improvements.

### Auto-Detect GPU Hardware (#434)
- **Author:** Ramon Reichert
- Automatic GPU detection eliminates the need for manual `KRONK_PROCESSOR` configuration
- Intelligent hardware detection for both AMD and Intel Macs
- Added exception handling for Intel Macs to ensure proper CPU fallback

**Benefits:**
- Simplified setup for new users
- Reduced configuration errors
- Better out-of-box experience for model downloads and server startup

## 🔄 Upgrade Notes

### Breaking Changes

#### Default Port Change
The server default port has changed from **11430** to **11435**.

**Migration:**
If you have existing configurations referencing the old port, update them:

```bash
# Old port (11430)
kronk server start --port 11430

# New port (11435)
kronk server start --port 11435
```

Or update any client configurations, reverse proxies, or firewall rules to use port 11435.

### Migration
No migration required for most users. The server supports explicit port configuration via the `--port` flag.

### Recommended Actions
- Update any client configurations to use the new default port (11435)
- Review firewall rules if port 11430 was explicitly allowed
- Consider removing manual `KRONK_PROCESSOR` environment variable as GPU detection is now automatic
- Verify Nix builds are using the new gomod2nix configuration
