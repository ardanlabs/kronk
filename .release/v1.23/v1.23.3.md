# Release Notes - v1.23.3

**Release Date:** April 22, 2026

## Overview

Version 1.23.3 introduces a major refactoring of the model processor to support a wide variety of tool-calling formats, along with new MCP tools, IMC improvements, and documentation updates.

## Detailed Changes

### New Features

#### New MCP tool for fuzzy replace (#500)

- **Author:** William Kennedy
- Added a new MCP tool for fuzzy replacement to the system.

### Improvements

#### Refactored tool calling processor (#504)

- **Author:** William Kennedy
- Completely rewrote the model processor architecture to support multiple tool-calling formats (JSON, Mistral, Gemma, Qwen, and GLM) using specialized parser implementations.
- Improved extensibility for adding new model formats.

#### IMC fixes (#500)

- **Author:** William Kennedy
- Implemented fixes for the Incremental Message Cache (IMC) to improve reliability.

### Bug Fixes

#### New jsonrepair fix (#503)

- **Author:** William Kennedy
- Fixed an issue where jsonrepair could fail during tool call processing.

### Documentation

#### Agent support documentation updates (#501)

- **Author:** William Kennedy
- Updated and clarified documentation regarding agent support.
