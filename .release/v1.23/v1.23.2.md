# Release Notes - v1.23.2

**Release Date:** April 21, 2026

## Overview

Version 1.23.2 includes fixes for llama.cpp compatibility and several improvements to the Incremental Message Cache (IMC) and JSON repair capabilities.

## Detailed Changes

### Improvements

#### JSON Repair Enhancements (#494)

- Enhanced the jsonrepair mechanism for better robustness during conversation rebuilding.

#### IMC Semantics Improvement (#496)

- Refined the Incremental Message Cache (IMC) with improved semantics for more accurate cache matches.

### Bug Fixes

#### llama.cpp Compatibility (#498)

- Fixed an issue where the removal of `ModelParamsFit` in recent llama.cpp versions caused failures.

#### IMC and JSON Repair Fixes (#495, #497)

- Fixed bugs related to IMC document handling and JSON repair implementation.

### Documentation

#### IMC Manual Update (#497)

- Updated documentation regarding IMC to ensure accuracy with recent changes.
